1. profil-utama.jpg
   Foto utama di hero section. Rekomendasi: portrait profesional, rasio 4:5.

2. project-01-ai-classification.jpg
   Gambar untuk AI Classification Research. Bisa berupa diagram model, dataset preview, dashboard hasil, atau poster riset.

3. project-02-serenade-analytics.jpg
   Gambar untuk Serenade Decor Analytics. Bisa berupa dashboard analytics, chart engagement, atau screenshot campaign.

4. project-03-helio-branding.jpg
   Gambar untuk PT Helio Digital Branding. Bisa berupa product photoshoot, katalog digital, atau contoh visual branding.

5. project-04-dormitory-community.jpg
   Gambar untuk Dormitory Community Development. Bisa berupa mentoring session, kegiatan asrama, atau dokumentasi peserta.

6. project-05-genbi-leadership.jpg
   Gambar untuk GenBI Leadership Initiative. Bisa berupa kegiatan GenBI, leadership program, partnership event, atau dokumentasi komunitas.

7. profil-kontak.jpg
   Foto bagian kontak. Rekomendasi: foto casual-professional yang hangat.

Catatan:
- Nama file harus sama persis.
- Format yang paling aman: .jpg
- Jika ingin pakai .png, ubah juga src gambar di HTML dari .jpg ke .png.
- Folder foto harus berada satu folder dengan file HTML, seperti ini:
  Mario_Reggy_Yedija_Apple_Portfolio_Modified.html
